# MazeTrix_Capstone

/main Contains the most up-to-date finished developments
The Unity build and Firebase setup are made to work on WINDOWS devices only.

Running instructions for final Demo:
1. Download the Final v1.0.2 zip.zip from /main/Build Zips
2. Extract the contents
3. Run the MazeTrix_Capstone.exe within the folder
4. Register an account
5. Navigate to the Login and enter credentials
6. START
7. To exit, press ESC to bring up the Pause menu and click ESC to ensure data is saved properly.
